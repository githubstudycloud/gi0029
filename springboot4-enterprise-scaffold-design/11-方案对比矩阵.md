## 矩阵格式说明

| 列 | 含义 |
|----|------|
| ID | 决策编号（与各文档章节对齐） |
| 决策点 | 要拍板的事项 |
| 候选方案 | A / B / C ... |
| 优势 | 该方案优点 |
| 劣势 | 该方案缺点 |
| 推荐 | ✅ 表示倾向，⚠️ 表示备选，❌ 不推荐 |
| 理由 | 推荐的简要原因 |
