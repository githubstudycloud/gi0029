# 参考资料与版本依据

> 访问日期：2026-05-06

## 1. Spring Boot 4

- Spring Boot 官方系统要求显示，当前 4.0.x 要求至少 Java 17，并兼容到 Java 26；本设计选择 JDK 21 是企业 LTS 策略，不是框架最低要求。  
  来源：https://docs.spring.io/spring-boot/system-requirements.html

- Spring Boot 4.0 release notes 提醒从 3.5 升级到 4.0 属于大版本迁移，需要参考迁移指南。  
  来源：https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Release-Notes

## 2. Spring Cloud 与 Gateway

- Spring Cloud 官方项目页显示 2025.1.x release train 对应 Spring Boot 4.0.x。  
  来源：https://spring.io/projects/spring-cloud

- Spring Cloud Gateway 5.0.1 文档显示它基于 Spring Framework 7、Spring Boot 4 和 Project Reactor，并提供 WebFlux/Web MVC 两种 Server 形态。  
  来源：https://docs.spring.io/spring-cloud-gateway/reference/index.html

- Spring Cloud 2025.1.1 发布说明提到 2025.1.x release train 兼容 Spring Boot 4.0.1 及后续 4.0.x。  
  来源：https://spring.io/blog/2026/01/29/spring-cloud-2025-1-1-aka-oakwood-has-been-released

## 3. Spring AI

- Spring AI 当前稳定文档说明 1.x 支持 Spring Boot 3.4.x 和 3.5.x，因此不应把 Spring AI 1.x 直接作为 Boot 4 默认基线。  
  来源：https://docs.spring.io/spring-ai/reference/getting-started.html

- Spring AI 2.0.0-M1 发布说明提到 2.x 线基于 Spring Boot 4、Spring Framework 7 和 Jakarta EE 11。  
  来源：https://spring.io/blog/2025/12/11/spring-ai-2-0-0-M1-available-now

- Spring AI 2026-04-27 发布说明显示 1.0.6、1.1.5、2.0.0-M5 已发布；其中 2.0.x 仍需按里程碑/后续 GA 策略验证。  
  来源：https://spring.io/blog/2026/04/27/spring-ai-1-0-6-1-1-5-2-0-0-M5-available-now

## 4. Spring Cloud Alibaba / Nacos

- Spring Cloud Alibaba 2025.1.0.0 release notes 显示支持 Spring Boot 4.x 和 Spring Cloud 2025.1.x，并且 Nacos 配置移除 bootstrap config，改用 `spring.config.import`。  
  来源：https://github.com/alibaba/spring-cloud-alibaba/releases

## 5. ShardingSphere

- ShardingSphere 5.3.x Spring 配置更新说明提到 Spring Boot Starter 和 Spring Namespace 会受到 Spring 更新影响，并带来配置兼容问题；因此本设计把 ShardingSphere 从默认依赖降为可选 starter。  
  来源：https://shardingsphere.apache.org/blog/en/material/2023_03_07_shardingsphere_5.3.x_update_spring_configuration_update_guide/
